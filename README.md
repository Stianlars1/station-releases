# Station

**The local development command center for Mac.**

Station runs, inspects, and safely repairs your whole local stack — services,
groups, ports, Docker, terminals, and dependency maps — in one native Mac app.
When something breaks, its governed AI Doctor explains why and proposes repairs
you approve, one at a time.

This repository hosts Station's public release artifacts (notarized DMGs,
`release.json`, and the Sparkle appcast) alongside the changelog.

## Download

**[Download the latest release for Mac →](https://usestation.app/download/latest)**

That link always serves the newest notarized build. To grab a specific version,
use the [Releases](https://github.com/Stianlars1/station-releases/releases) page.

## Requirements

- macOS 14 (Sonoma) or newer
- Apple silicon or Intel
- Local-first — no account required to start the 7-day trial
